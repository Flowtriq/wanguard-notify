# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email **security@flowtriq.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

We will acknowledge receipt within 48 hours and provide a detailed response within 5 business days.

For more details, see our [Responsible Disclosure Policy](https://flowtriq.com/security).

## Supported Versions

We provide security updates for the latest release only. Please keep your installation up to date.
